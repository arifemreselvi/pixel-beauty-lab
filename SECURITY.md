# Security Policy

## Supported Versions

This project is actively maintained on the latest default branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report details privately to the maintainer with:

- vulnerability description
- impact assessment
- reproduction steps
- potential fix (if known)

You should receive an acknowledgment within 72 hours.

## Scope Notes

- This is a local desktop app and does not include a hosted backend.
- Be cautious with third-party model files; only use trusted sources.
- Avoid committing sensitive local files such as logs, environment files, or credentials.
