# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in our project
and our community a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the project

Examples of unacceptable behavior:

- Harassment, insults, or discriminatory language
- Personal attacks or trolling
- Public or private intimidation

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.
They may remove, edit, or reject comments, commits, code, and other contributions
that are not aligned with this code of conduct.

## Scope

This code of conduct applies within all project spaces and in public spaces when
an individual is representing the project or its community.
